## 🎯 **Grading System**

</div>

| Grade | Stars | Description | Color |
|-------|-------|-------------|-------|
| **PERFECT** | ⭐⭐⭐⭐⭐ | Complete, accurate, all key points covered | 🟢 Green |
| **GOOD** | ⭐⭐⭐⭐ | Most concepts correct, minor gaps | 🔵 Blue |
| **PARTIAL** | ⭐⭐⭐ | Genuine partial understanding | 🟡 Orange |
| **WEAK** | ⭐⭐ | Major gaps or errors | 🔴 Red |
| **WRONG** | ⭐ | Incorrect or contradicts reference | 🔴 Red |

**Smart Evaluation:**
- ✅ Detects question types (calculation/factual/definition/explanation)
- ✅ Doesn't penalize for missing unrequested information
- ✅ Catches contradictions (marks WRONG, not PARTIAL)
- ✅ Verifies against actual course material
- ✅ Fair grading with no false negatives

## 🔧 **Troubleshooting**

</div>

<details>
<summary><b>⚠️ Common Issues & Solutions (Click to Expand)</b></summary>

<br/>

### 🔴 **Ollama Not Found**

```bash
# Install Ollama manually
brew install ollama
brew services start ollama
ollama pull llama3.2:3b
```

### 🔴 **Model Not Responding**

```bash
# Check Ollama status
ollama list    # See installed models
ollama ps      # See what's running
```

### 🔴 **Python Version Issues**

```bash
# Install Python 3.11+
brew install python@3.11 python-tk@3.11
```

### 🔴 **macOS Version Error**

✅ **Fixed!** The app now handles macOS 26 (Sequoia) correctly.

### 🟢 **Reset Everything**

```bash
brew services restart ollama
./start_quiz.sh
```

</details>
